# Restorevscodefiles
Recovering old files from the File History directory AppData

Данный файл позволит считать из системной папки Visal Code Studio автоматически сохраненые файлы. 

Случайно стер важные файлы проекта, узнал, что в папке c:\Users\ИМЯПОЛЬЗОВАТЕЛЯ\AppData\Roaming\Code\User\History\ хранятся копии файлов, попросил Claude Code создать панельку, которая считает json файлы и выведет в виде панели для возможности скачивания. 

Очень удобно, только нужно заменить путь в файле.
